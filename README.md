# Cardsprint – Next.js Shop

Ein kleiner, fokussierter Shop (Next.js, Pages Router) mit Anbindung an **Paysafe** (Paysafecard) und **Stripe** (Karte/Apple Pay/Google Pay). Deployment über **Vercel**.

---

## 🚀 Projektüberblick
- **Framework:** Next.js 14 (Pages Router)
- **Hosting:** Vercel
- **Zahlungen:** Paysafe (Paysafecard) + Stripe (Kreditkarte, Apple Pay/Google Pay – falls im Stripe-Account aktiviert)
- **Styles:** Tailwind CSS

---

## 📦 Tech-Stack
- **Next.js 14**
- **TypeScript**
- **Tailwind CSS**
- **API-Routen** unter `pages/api/*` für serverseitige Payment-Logik
